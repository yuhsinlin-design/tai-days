# Tâi Days｜日日台灣
**A small piece of Taiwan, every day.**

Tâi Days is a daily calendar inspired by the tear-off calendars found in Taiwanese homes and shops.

Each day introduces a Taiwanese (Tâi-gí) word alongside its Tâi-lô pronunciation, meaning, example sentence, and cultural context. Throughout the year, entries are connected to seasonal traditions, festivals, historical moments, and everyday experiences that shape life in Taiwan.

Designed as a quiet daily ritual, Tâi Days brings a small piece of Taiwan to your desktop every day.

---

## What You'll Discover

### Language

* Taiwanese Hanji
* Tâi-lô pronunciation
* Mandarin explanation
* Example sentence
* Vocabulary audio (sourced from Moedict)

### Seasons & Traditions

* Lunar calendar dates
* Twenty-four Solar Terms (二十四節氣)
* Traditional festivals
* Seasonal foods and customs

### Taiwan Stories

Selected entries throughout the year highlight:

* Historical events
* Cultural movements
* Seasonal traditions
* Everyday moments

that continue to shape life and memory in Taiwan.

## Data Structure

Each calendar entry contains:

| Field            | Description                               |
| ---------------- | ----------------------------------------- |
| Date             | Gregorian calendar date                   |
| Lunar Date       | Lunar calendar information                |
| Hanji            | Taiwanese characters                      |
| Tâi-lô           | Romanized pronunciation                   |
| Mandarin         | Chinese explanation                       |
| Example Sentence | Taiwanese example sentence                |
| Translation      | Mandarin translation                      |
| Audio            | Vocabulary pronunciation                  |
| Theme            | Seasonal, cultural, or historical context |

## About

Tâi Days began as a personal project to explore how language can serve as a gateway to culture.
By bringing together Taiwanese language, seasonal traditions, historical moments, and everyday stories, the project aims to create a small daily connection to Taiwan—whether you're living on the island or halfway around the world.
